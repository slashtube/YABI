# YABI
YABI (Yet Another Brainfuck Interpreter) is (guess what) a brainfuck interpreter written in C.
## Usage
Compile the code

`gcc main.c -o YABI`

Run `./YABI "[Code]"`

I tested this only on Linux.
## Example
`./YABI "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++."`

Outputs: `Hello World!`
